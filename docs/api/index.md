# API Reference

This section contains the API reference for `@addmaple/stats`.

## Where are the function pages?

The detailed function reference pages are generated from the JS source.

- If you’re browsing the published site, use the sidebar under **API Reference**.
- If you’re building locally, run:
  - `npm run docs:api` (TypeDoc output)
  - `npm run docs:functions` (per-function pages)
