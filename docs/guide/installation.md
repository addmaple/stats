# Installation

## npm

```bash
npm install @stats/core
```

## yarn

```bash
yarn add @stats/core
```

## pnpm

```bash
pnpm add @stats/core
```

## Requirements

- Node.js 18+ (for ESM support)
- Modern browser with WebAssembly support (all modern browsers)

## TypeScript Support

TypeScript definitions are included. No additional `@types` package needed.

```ts
import { init, mean } from '@stats/core';
```
